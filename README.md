# Wirless motion sensor

An arduino based wireless PIR sensor

## Parts list

* Arduino pro mini
* NRF24L01+ wireless transciever
* HC-SR501 PIR sensor
* 14500 Lithium-ion battery
* LED + resistor
* Power switch
* 5.5mm x 2.1mm DC socket
* 2P JST connector for charge
